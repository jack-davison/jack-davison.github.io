Air Quality on Bonfire Night 2023

professional
quarto
air-quality
Perhaps unsurprisingly, measured concentrations of airborne particulates typically peak on Bonfire Night in the UK. This quarto report examines bonfire night on 2023 in particular, and provides commentary on the utility of the “Daily Air Quality Index” (DAQI) as a framework for short-lived events like a bonfire event.

UK Summer 2023 Ozone

professional
quarto
air-quality
Typically, during the hottest parts of summer, the UK will experience episodes of elevated ozone which are associated with deleterious health effects. This quarto report summarises a particularly protracted ozone episode in 2023, and includes applications of then-new functionality of {gt} and {bslib}.
19-10-2023
Jack Davison, Ricardo

I am an R developer with experience in package and Shiny app development. All of my public code can be found on my GitHub profile, but this section outlines a few examples.

{openair}

r-package
openair-toolkit
air-quality
{openair} is a long standing R package for atmospheric composition data analysis. I was added as an author over a decade into its existence and have made efforts to modernise some of the code, fix back-logs of issues, and add features with a focus on making it easier to program with {openair}.

David C. Carslaw, Jack Davison, Karl Ropkins

{openairmaps}

r-package
openair-toolkit
air-quality
{openairmaps} combines {openair} analysis approaches with the {leaflet} interactive mapping library. The headline feature is the ability to put bivariate polar plots on interactive maps, which allows for pollutant source triangulation. Other features include visualising UK monitoring networks and HYSPLIT trajectory paths.

Jack Davison, David C. Carslaw

{lipsum}

r-package
{lipsum} is a very simple package for calling a variety of Web APIs from the awesome-lipsum list. It was mainly an exercise in playing with the {curl} package, but could be useful for users to obtain a variety of fun placeholder texts.

Jack Davison

Using {gt}, {gtExtras} and {openair} to present air quality monitoring data

quarto
contest
table-contest
A submission to the Posit (then RStudio) Table Contest 2021; a tutorial for presenting air quality monitoring data in summary tables using {gt}. The most interesting aspect of this was combining {openair} with {gt} to embed bivariate polar plots in the {gt} table itself. Awarded “Honourable Mention”.
